## 開発言語
---
- Ruby 3.0.1
- Ruby on Rails 6.0.3

## 就業Termの技術
---
- device
- Ajaxを使ったコメント機能
- フォロー機能
- お気に入り機能

## カリキュラム外の技術
---
- gem ransackによる検索機能 

## 実行手順
---

```
$ git clone git@github.com:KeiNakamu/share_housework_app.git
$ cd share_housework_app
$ bundle
$ rails db:create && rails db:migrate
$ rails s
```
